I/ FONCTIONNEMENT DE L'APPLICATION

Mon application utilise la technologie du WebSocket qui a les avantages de la rapidité de transfert de données client/serveur,
et la mise à jour automatique des données du site (comme le nombre de points par exemple).
Les mots sont stockés dans une base de données MongoDB.

II/ INSTALLATION DE L'APPLICATION

1) Après avoir installé npm , lancer la commande : "npm install"
2) Executer la commande depuis un terminal : "node index.js"
3) Le serveur de jeu sera complètement lancé après l'affichage du message "Les mots ont bien été chargé le serveur est lancé !"
3) Accéder à l'URL http://localhost:9000 depuis un navigateur
